podiceps - a simple program for maintaining a dictionary of foreign words.

The project was created:
- to study the C++ programming language(without STL and standards);
- to master the creation skill a GUI(with FLTK);
- to master the creation skill the unit tests(with CppUTest);
- ...

It is possible to work through the command line and through the GUI.
